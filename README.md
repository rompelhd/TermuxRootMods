<h2 align="center">ğŸ“¦ What is TermuxRootMods?</h2>

<p align="center">
TermuxRootMods is a <strong>Magisk module</strong> that enhances the Termux experience on <strong>rooted Android devices</strong>.
It provides a set of lightweight utilities that integrate root access directly into Termux, without modifying the global Android system.
</p>

<p align="center">
